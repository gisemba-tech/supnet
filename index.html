What I Wish I Knew Before Starting Web Development.

1.You don't need to learn everything at once. Get overwhelmed by HTML, CSS, JS, React, Node, Git, etc.Start with the fundamentals-donâ€™t try to become a full- stack developer overnight.

2.Javascript is Your Best Friend (and Worst Enemy) as it powers most of the web-invest time learning it deeply. It also has quirks-like undefined, == vs ===, and scope wierdness

Top 4 Free Tools Every Beginner Web Developer Should Use.

1.VS Code (text editor)- A powerful Microsoft code editor,with library of extensions, built-in Git support,intelligent code completion(IntelliSense), and integrated terminal

2.GitHub (version control)- Is a version control system used for hosting and collaborating on repositories. It keeps track of code changes,enables collaboration and supports CI/CD pipelines.

3.Google Fonts (typography)- A free,open-source library of web fonts provided by Google.It offers a huge collection of typefaces that can be easily integrated into websites.

4.Figma (basic design)- A collaborative design tool for UI/UX used by developers to inspect designs,export assets,and collaborate with designers in real-time.
